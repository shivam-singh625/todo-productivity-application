# Screenshots

Place your app screenshots here:

| Filename | Content |
|----------|---------|
| `dashboard.png` | Dashboard with stats, streaks, charts |
| `tasks.png` | Task list with date grouping |
| `analysis.png` | Test analysis panel |
| `calendar.png` | Calendar view |
| `timer.png` | Floating timer widget |
| `pomodoro.png` | Pomodoro sidebar |

## How to take screenshots on Linux

```bash
# Full window screenshot (XFCE)
xfce4-screenshooter -w

# Or use scrot
scrot -u docs/screenshots/dashboard.png
```
